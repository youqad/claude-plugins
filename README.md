# claude-plugins

Claude Code plugins. Install as a marketplace:

```bash
claude plugin marketplace add https://github.com/youqad/claude-plugins
claude plugin install claude-code-deep-explanatory-output-style@claude-plugins
```

## Plugins

- **[claude-code-deep-explanatory-output-style](claude-code-deep-explanatory-output-style/)**: first-principles pedagogical insights (replaces the built-in explanatory output style)
